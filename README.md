The Redmi 9C NFC (codenamed _"angelican"_) are mid-range smartphones from Xiaomi.

## Device Specifications

| Basic                   | Spec                                                                         |
| ----------------------- | :--------------------------------------------------------------------------- |
| SoC                     | MediaTek Helio G35 (MT6765G) (12nm)                                          |
| CPU                     | 4 x 1.8 GHz Cortex-A53 & 4 x 2.3 GHz Cortex-A53                              |
| GPU                     | PowerVR GE8320                                                               |
| Memory                  | 2GB / 3GB / 4GB                                                              |
| Shipped Android version | 10                                                                           |
| Storage                 | 32GB / 64GB / 128GB                                                          |
| Battery                 | Non-removable Li-Po 5000 mAh (10W slow charging)                             |
| Dimensions              | 164.9 x 77.1 x 9 mm (6.49 x 3.04 x 0.35 in)                                  |
| Display Type            | IPS LCD, 400 nits (typ) 							 |
| Display Size            | 6.53 inches, 102.9 cm2 (~81.0% screen-to-body ratio)                         |
| Display Resolution      | 720 x 1600 pixels, 20:9 ratio (~269 ppi density)                             |
| Rear Camera             | 13 MP, f/2.2, 28mm (wide), 1.0µm, PDAF 2 MP (macro)                          |
| Front Camera            | 5 MP, f/2.2, (wide), 1.12µm                                                  |
